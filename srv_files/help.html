AI-Inspector Server Help Page

Available GET requests:

  • /help: Returns this page
    • Input: None
    • Return: HTML with this help message"
  • /getstatus: Returns a JSON file with current internal state of the AI-Inspector
    • Input: None
    • Return: JSON file with status information"
  • /setidle: Switches back to idle state.
    • Input: None
    • Return: JSON file with status information"

Available POST requests:

  • /settrain: Starts the training of an application.
    • Input: JSON file with the application name
    • Return: Updates the status command with the progress and current accuracy
  • /setinference: Starts the inference of an application.
    • Input: JSON file with the application name
    • Return: JSON file with: "tag_last_inference", "inference_result" and "inference_time"
